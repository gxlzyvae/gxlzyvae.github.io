Markdown入门教程

发表于2022-04-22|更新于2022-04-23|hexo
|字数总计:808|阅读时长:2分钟|阅读量:

  • 前言

    Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，在2004年由约翰·格鲁伯创建。Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用。看到这里请不要被「标记」、「语言」所迷惑，Markdown 的语法十分简单。常用的标记符号也不超过十个，这种相对于更为复杂的 HTML 标记语言来说，Markdown 可谓是十分轻量的，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。

  • 学的不仅是技术，更是梦想

    我记得我以前上课的时候老师说过：你们要有什么不会的上菜鸟。然后他就走了，留给我们”英俊“的背影…！上菜鸟开始你的编程之路吧，一学就废那种！当然你也可以在我这里学！ (哈哈哈) Markdown-菜鸟

  • 编辑器

    关于同什么编写Markdown，我推荐使用VS code和Typore这两款软件，因为它们都比较轻便易用，对新手较为友好。附上下载地址： VS code Typore不是官网，但无毒，无捆绑 Typore

  • Markdown 标题

    标题分为：一级标题 二级标题 三级标题 四级标题 五级标题 六级标题 六种。 Markdown有两种。第一种： = 和 - 标记语法格式 ，第二种：以 # 号代表表标题等级，#为一级标题，##为二级标题，以此类推。

  • Markdown 换行 列表

    换行： Markdown的换行非常简单，按 两次空格 在加 回车键 就可以实现换行。特别提醒一点： 书写Markdoen的格式时记得在 内容前后和后面加一个空格 在书写内容，不然格式不显示！

    • 无序列表： 只需要在内容前面加上一个星号(*)即可。
    • 有序列表： 只需要在内容前面加上一个星号(.)即可。示例：1. 2. 3.

  • Markdown 字体格式 分割线

    • Markdown 提供了3种字体格式，每种格式有两种书写格式，下面只用最简单的书写格式。示例：（斜体文本） （粗体文本） （粗斜体文本）
    • 分割线：你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。星号分割线（1.*** 2.* * * 3.*****）减号分割线（1.- - - 2.———-）

  • Markdown 添加链接

    Markdown提供了两种链接的方式，第一种：方括号(链接地址) ，第二种：<链接地址>。

  • Markdown 添加图片

    开头一个感叹号 ! 加一个方括号，方括号里面填写描述图片的文字，方括号后面加一个括号(),附上图片地址即可！

  • 总结

    Markdown还有一些内容没写上，因为我觉得用不上，以上这些是我觉得可以用的上的，希望能帮上你。推荐一位大神写的更好topking 再见！ 瑞思拜
