butterfly主题优化

发表于2022-04-23|更新于2022-04-24|hexo
|字数总计:1.5k|阅读时长:5分钟|阅读量:

  • 前言

    hutterfly是我觉得非常适合作为个人博客的主题，关于butterfly主题优化网上有很多的教程，但有些还是计较难懂的，我也懒得去解析和摸索了，简单为主，且butterfly主题里的备注多为繁体字，对于我们配置主题多少有点困扰，但经过我的摸索和网上教程的辅助，我终于完成了主题的配置，废话不多，开始！

  • 安装butterfly主题

    进到你的hexo博客themes文件夹下运行 git Bash 输入：(这里推荐的是稳定版)

  • 安装渲染器

    安装了渲染器，主题才会渲染到你的页面上，不然你运行了hexo，页面上也是空白一遍！在hexo博客根目录下运行 git Bash 输入：

  • 应用主题

    修改 Hexo 根目录下的 _config.yml，找到关键字：theme 把后面的值改为：butterfly

    （之前我做了如上操作后，我运行了hexo，但页面还是没有内容，如果遇到像我这样的情况请如下操作：你可以看到butterfly主题是嵌套在一个文件夹下的，嵌套butterfly主题的文件夹叫什么名字我忘了，顺序是：<文件夹> butterfly ，你需要做的是把butterfly文件夹从那个文件夹中剪切出来放在theme文件夹下，之后运行 hexo s 后页面看看是不是有效果！）

  • 升级建议

    这个升级建议的就是在你的hexo根目录下创建一个_config.butterfly.yml 文件，并把主题里的_config.yml文件里的内容复制过来，以后修改主题配置的时候就可以在 _config.butterfly.yml 文件下进行即可。

  • 主题基础配置

  • 主题基础配置我参考的是一位叫< 其实不会敲代码>大佬的教程，我搜遍了网络上的教程，认为这篇是讲得比较基础也比较详细易懂的，仔细看完他的教程，你的主题优化的已经基本成型了，剩下的就是一些个性化的设置了！

    当然我分享一些我在优化主题过程中遇到一些问题的解决方式告诉大家，以免少走一些弯路！

  • 导航栏配置

    备注1：||——双竖杠后面的代表的是导航栏的图标

    备注2：如果想创建二级菜单，记得缩进的距离一定要一致

    备注3：一定要英文的冒号: 冒号后面记得加个空格在输入内容

    备注4：被双斜杠包住的是文件的名称及路径，如果你想在导航栏添加自己的内容，只需要创建一个文件，并在双斜杠中间添加你文件的名字就可以，记得文件名要是英文；创建文件的命令：hexo n page 文件名

  • 添加音乐

    添加全局吸底音乐播放器，这个有个问题我一直没有解决，就是切换页面的时候，音乐会停止，而且到了新页面，音乐可能就会自动播放，我实在搞不懂!但起码有音乐了这才是最重要的！我看了官方的讲了一大推比较难懂，其实直接按照我下面的操作也可以实现！官方教程

    1.开安装插件，在你的根目录下运行 git bash 安装即可： npm install --save hexo-tag-aplayer

    2.在主题配置文件里，搜索关键字：aplayerInject 将下面的值设置为true

    3.在主题配置文件中搜索关键字：inject , 在bottom值下添加链接。想要添加自己的喜欢歌单时，只要改变下面链接中： data_id=””的值即可！

  • 网页字体&字体大小

    打开主题配置文件，搜索关键字：font 修改下面值的数据即可！

  • 评论系统

    我主题应用的是valine评论系统，只需要注册valine的账号，并把valine的app_id+appkye添加到主题配置中即可！这是我认为说的比较好的教程，献给大家！传送门

    教程只是让你知道该怎么做，实际操作中还是会出现一些问题，下面展示的是主题配置评论系统的两个关键节点，只需要做好这两个配置操作你的评论系统就会出现！(valine评论系统)

    一、搜索关键字 comments 并在use 下填写评论系统的名字 （valine）

    二、搜索关键字 valine 添加valone评论系统的appId+appkey,完成这两步，重启hexo博客，你的文章就会出现评论区了！
